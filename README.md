# Calculadora Simples em React Native (Tema Hieróglifos)

Este é um projeto de aplicativo móvel desenvolvido como uma **Calculadora Simples** utilizando **React Native**, **Expo** e a biblioteca de componentes visuais **React Native Paper**.

O grande diferencial desta calculadora é que tanto a entrada de dados (botões numéricos) quanto o resultado no visor são exibidos usando **Hieróglifos Egípcios** (mapeamento representativo dos dígitos).

---

## 📱 Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Native Paper](https://reactnativepaper.com/) (Componentes visuais no padrão Material Design)

---

## ✨ Funcionalidades

- Operações matemáticas básicas: Adição (`+`), Subtração (`-`), Multiplicação (`*`) e Divisão (`/`).
- Botão para limpar a tela (`C`).
- Visor e botões convertidos automaticamente para hieróglifos (Exemplo: `1` = `𓏺`, `2` = `𓏻`, `3` = `𓏼`, etc).
- Layout responsivo adaptado com tema egípcio utilizando a biblioteca do React Native Paper.

## 🛠️ Como Instalar e Rodar o Projeto Localmente

### Pré-requisitos
- Ter o **Node.js** instalado na sua máquina.
- Ter o app **Expo Go** instalado no seu celular, ou algum emulador Android/iOS configurado no computador.

### Passo a passo

1. **Clone o repositório** para a sua máquina ou baixe o código.
2. Abra o terminal na pasta do projeto.
3. Instale as dependências executando o comando:
   ```bash
   npm install
   ```
4. Para iniciar o servidor de desenvolvimento do Expo, execute:
   ```bash
   npx expo start
   ```
5. Com o servidor rodando, você pode visualizar o App de diversas formas:
   - **Celular (Físico):** Abra o aplicativo "Expo Go" e escaneie o QR Code que apareceu no terminal.
   - **Emulador:** Pressione a tecla `a` (para emulador Android) ou `i` (para emulador iOS) no terminal onde o Expo está rodando.
